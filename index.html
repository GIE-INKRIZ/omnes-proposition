① Modules fonctionnels & données monitorées
Module 1 — Visibilité IA / GEO
Intégration aux dashboards GEO existants d'Omnes Education
Connexion API / webhooks — Branchement sur les dashboards GEO déjà opérationnels
Share of Voice — Omnes vs concurrents dans les LLMs (%)
Sentiment — Score 0–100 sur les réponses IA
Geo — Visibilité par zone géographique
Sources — Domaines cités (media / instit / blog)
Keywords / Clusters — Perf par mot-clé + évolution
Concurrents — Mentions dans les réponses IA
Bot visits / page — Crawls IA sur les sites Omnes
Prompts — Requêtes TOFU / MOFU / BOFU
Source : Dashboards GEO existants du client — aucun moteur redéveloppé from scratch
Module 2 — Automatisation Réputation
Pipeline avis · Reddit Q&A · Interface modération & réponses IA
Google Avis — Collecte, score moyen, évolution par école
Trustpilot — Volume, rating, verbatims récents
Diplomeo / Custplace — Agrégation avis secteur ESR
Reddit Q&A — Détection questions/mentions par école
Interface de modération — Flag des avis par plateforme directement dans le dashboard
Réponses pré-générées — Suggestions de réponse par LLM, envoi depuis la plateforme
Score réputation — Agrégat multi-plateforme par entité
Sources : Google Places API, Trustpilot API, scraping Diplomeo/Custplace, Reddit API
Module 3 — Performance Marketing
GA4 · Google Ads · Meta Ads · Saisie interne
Visites humaines — Trafic par site (sessions, pages vues)
Budget — Saisie manuelle ou import CSV par période
Google Ads — CPC, impressions, conversions, ROAS
Meta Ads — CPC, reach, CPL, taux de conversion
Salons — Présences, leads générés, ROI estimé
Sources : Google Analytics 4, Google Ads API, Meta Ads API, saisie interne
Module 4 — Social Media
APIs Meta · LinkedIn · TikTok · YouTube · Google Trends
Publications — Volume, fréquence, contenu par réseau
Performance réseaux — Reach, engagement, followers
Facebook / Instagram — Métriques Meta Graph API
LinkedIn — Impressions, clics, followers page
TikTok — Vues, engagement, croissance
YouTube — Abonnés, vues, durée de visionnage, engagement, performance par vidéo & par école
Tendances — Hashtags & sujets émergents ESR
Sources : Meta Graph API, LinkedIn API, TikTok API, YouTube Data API v3, Google Trends
